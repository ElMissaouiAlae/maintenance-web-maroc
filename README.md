# Maintenance Web Maroc

Landing page professionnelle pour un service de maintenance web destiné aux petites entreprises locales au Maroc.

## Objectif

Ce site présente une offre simple de maintenance, correction et amélioration de sites web pour les entreprises à Rabat, Temara et Kénitra.

## Services présentés

- Corrections de bugs
- Maintenance web mensuelle
- Amélioration mobile
- Ajout de bouton WhatsApp
- Vérification des formulaires
- SEO local basique
- Optimisation de pages existantes

## Technologies utilisées

- React
- Vite
- Tailwind CSS
- Lucide React
